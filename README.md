# MovieMania

MovieMania är en enkel webbplats öfr personer som tycker om anime, serier och underhållning.  
Webbplatsen innehåller ett bildgalleri, navigationsmeny, formulär och flera sidor byggda med HTML och CSS.

## Målgrupp
Webbplatsen riktar sig till alla som gillar anime utan specifik åldersgrupp


### **HTML** 
- Semantiska element används (header, nav, main, footer) 
- Flera undersidor finns (Home, Contact, Login, Create Account, More) 
- Bilder med alt‑texter 
- Navigationsmeny som fungerar på alla sidor 
- Formulär på kontaktsidan / login / skapa konto


### **CSS** - 
- Extern CSS‑fil 
- Flexbox och Grid används 
- Responsiv design med media queries (mobil, tablet, desktop)
- Färger, knappar och layout är konsekventa 
- Box‑shadow, border‑radius och spacing för modern design


### **Övrigt** 
- Projektet är strukturerat i mappar (t.ex. images/, styles/) -



### **Kör lokalt** 
1. Ladda ner projektmappen 
2. Öppna `index.html` i valfri webbläsare 
3. Navigera mellan sidorna via menyn
